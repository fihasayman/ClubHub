# ClubHub – University Club Management System

## Overview

ClubHub is a Windows-based desktop application developed using C# to provide a centralized platform for managing university club activities across multiple institutions. The system supports universities such as AIUB, BRAC University, North South University (NSU), and Independent University, Bangladesh (IUB), enabling clubs to operate through a unified and standardized workflow.

The primary objective of ClubHub is to streamline club administration by reducing manual processes and improving efficiency in managing members, events, volunteer activities, and club applications. The system ensures consistency across different universities while maintaining scalability and ease of maintenance.

## Features

### Role-Based Access Control

ClubHub implements a secure role-based access system with four user roles:

* **Admin**

  * Manage universities
  * Assign and manage club presidents
  * Monitor overall system operations

* **President**

  * Manage club activities
  * Organize and oversee events
  * Review member applications
  * Coordinate volunteer programs

* **Member**

  * Participate in club events
  * Register for volunteer activities
  * Access club-related information

* **User/Applicant**

  * Browse available clubs
  * View upcoming events
  * Submit club membership applications

## System Modules

### Authentication Module

Handles user login, access control, and session management.

### University Management Module

Allows administrators to manage universities and assign club presidents.

### Club Management Module

Supports club administration, membership management, and organizational activities.

### Event Management Module

Facilitates event creation, registration, participation tracking, and volunteer coordination.

### Application Management Module

Processes membership applications and approval workflows.

5. Build and run the project.
